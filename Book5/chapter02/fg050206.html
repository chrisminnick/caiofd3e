Table Components:
• <table> - The entire table element
• <thead> - Table header section (blue background)
• <tbody> - Table body section (gray/white rows)
• <tr> - Table row (each horizontal line)
• <th> - Table header cell (bold, in thead)
• <td> - Table data cell (regular content)
